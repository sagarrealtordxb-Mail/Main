# iOS Vulnerable Demo App - Source Code

This is a **intentionally vulnerable** iOS application created for security scanning and learning purposes.

## ⚠️ DO NOT USE IN PRODUCTION

This code demonstrates common security vulnerabilities found in iOS apps. Use this only for:
- ✅ Security training
- ✅ Vulnerability assessment
- ✅ Code review practice
- ✅ Security scanning demonstrations

---

## Files Included

1. **VulnerableApp.swift** - Main app with 10 common vulnerabilities
2. **VULNERABILITY_LIST.md** - Detailed vulnerability analysis
3. **FIXES_GUIDE.md** - How to fix each vulnerability

---

## Quick Start

```bash
git clone https://github.com/sagarrealtordxb-Mail/Main.git

```
